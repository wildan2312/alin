Tranformasi Matriks#

Transformasi matriks adalah operasi yang mengubah satu set data atau vektor menjadi set data atau vektor lain melalui penggunaan matriks. Dalam konteks aljabar linier, transformasi ini sering digunakan untuk melakukan berbagai operasi geometris seperti rotasi, skala, dan translasi pada vektor-vektor dalam ruang Euclidean.

Jenis-Jenis Transformasi Matriks dan Rumusnya#

1. Transformasi Identitas#

Transformasi identitas adalah transformasi yang tidak mengubah vektor. Matriks identitas biasanya dinotasikan dengan I.

Rumus:

\[\begin{split} [ I = \begin{pmatrix} 1 & 0 \\ 0 & 1 \end{pmatrix} ] \end{split}\]

2. Transformasi Skala (Scaling)#

Transformasi skala memperbesar atau memperkecil vektor dengan faktor tertentu.

Rumus:

\[\begin{split} [ S = \begin{pmatrix} s_x & 0 \\ 0 & s_y \end{pmatrix} ] \end{split}\]

Di mana (s_x) dan (s_y) adalah faktor skala sepanjang sumbu-x dan sumbu-y.

3. Transformasi Rotasi (Rotation)#

Transformasi rotasi memutar vektor sebesar sudut tertentu terhadap titik asal.

Rumus:

\[\begin{split} [ R(\theta) = \begin{pmatrix} \cos\theta & -\sin\theta \\ \sin\theta & \cos\theta \end{pmatrix} ] \end{split}\]

Di mana (\theta) adalah sudut rotasi.

4. Transformasi Translasi (Translation)#

Translasi menggeser vektor dari satu posisi ke posisi lain. Translasi biasanya tidak bisa direpresentasikan dengan matriks 2x2 biasa, tetapi bisa dengan menggunakan matriks homogen.

Rumus (Menggunakan Matriks Homogen 3x3):

\[\begin{split} [ T = \begin{pmatrix} 1 & 0 & t_x \\ 0 & 1 & t_y \\ 0 & 0 & 1 \end{pmatrix} ] \end{split}\]

Di mana (t_x) dan (t_y) adalah jarak translasi sepanjang sumbu-x dan sumbu-y.

5. Transformasi Refleksi (Reflection)#

Refleksi mencerminkan vektor terhadap sumbu tertentu.

Rumus (Refleksi terhadap sumbu-x):

\[\begin{split} [ F_x = \begin{pmatrix} 1 & 0 \\ 0 & -1 \end{pmatrix} ] \end{split}\]

Rumus (Refleksi terhadap sumbu-y):

\[\begin{split} [ F_y = \begin{pmatrix} -1 & 0 \\ 0 & 1 \end{pmatrix} ] \end{split}\]

6. Transformasi Geser (Shear)#

Transformasi geser mengubah bentuk vektor dengan menggesernya sepanjang sumbu-x atau sumbu-y.

Rumus (Shear horizontal):

\[\begin{split} [ Sh_x = \begin{pmatrix} 1 & k_x \\ 0 & 1 \end{pmatrix} ] \end{split}\]

Rumus (Shear vertikal):

\[\begin{split} [ Sh_y = \begin{pmatrix} 1 & 0 \\ k_y & 1 \end{pmatrix} ] \end{split}\]

Di mana (k_x) dan (k_y) adalah faktor geser horizontal dan vertikal.

Contoh Implementasi di Python (NumPy)#

import numpy as np

# Contoh vektor
v = np.array([1, 1])

# Matriks Rotasi 90 derajat
theta = np.radians(90)
R = np.array([[np.cos(theta), -np.sin(theta)], [np.sin(theta), np.cos(theta)]])

# Menerapkan transformasi rotasi
v_rotated = R @ v

print("Vektor asli:", v)
print("Vektor setelah rotasi 90 derajat:", v_rotated)

Vektor asli: [1 1]
Vektor setelah rotasi 90 derajat: [-1.  1.]

    ax.axhline(0, color='grey', linewidth=0.5)
    ax.axvline(0, color='grey', linewidth=0.5)
    ax.set_xlim(-lim, lim)
    ax.set_ylim(-lim, lim)
    ax.set_aspect('equal', adjustable='box')
    ax.grid(which='both', linestyle='--', linewidth=0.5)

# Fungsi untuk melakukan transformasi matriks
def transform_and_plot(matrix, points, ax, color='blue', label='Transformed'):
    transformed_points = matrix @ points
    ax.scatter(transformed_points[0, :], transformed_points[1, :], color=color, label=label)
    return transformed_points

# Definisi titik-titik (himpunan titik)
points = np.array([
    [1, 2, 3, 4, 5],  # X-koordinat
    [1, 2, 3, 4, 5]   # Y-koordinat
])

# Sudut rotasi dalam derajat
theta_degrees = 45
theta = np.radians(theta_degrees)

# Matriks transformasi untuk rotasi
rotation_matrix = np.array([
    [np.cos(theta), -np.sin(theta)],
    [np.sin(theta), np.cos(theta)]
])

# Plot hasil transformasi
fig, ax = plt.subplots()
draw_axes_and_grid(ax)
ax.scatter(points[0, :], points[1, :], color='red', label='Ori')
transformed_points = transform_and_plot(rotation_matrix, points, ax, color='blue', label=f'Rotasi {theta_degrees}')

ax.legend()

    return transformed_points

import numpy as np
import matplotlib.pyplot as plt

# Fungsi untuk menggambar sumbu dan grid
def draw_axes_and_grid(ax, lim=10):
    ax.axhline(0, color='grey', linewidth=0.5)
    ax.axvline(0, color='grey', linewidth=0.5)
    ax.set_xlim(-lim, lim)
    ax.set_ylim(-lim, lim)
    ax.set_aspect('equal', adjustable='box')
    ax.grid(which='both', linestyle='--', linewidth=0.5)

# Fungsi untuk melakukan transformasi matriks
def transform_and_plot(matrix, points, ax, color='blue', label='Transformed'):
    transformed_points = matrix @ points
    ax.scatter(transformed_points[0, :], transformed_points[1, :], color=color, label=label)
    return transformed_points

# Definisi titik-titik (himpunan titik)
points = np.array([
    [1, 2, 3, 4, 5],  # X-koordinat
    [1, 2, 3, 4, 5]   # Y-koordinat
])

# Faktor shearing
k = 1.5

# Matriks transformasi untuk shearing pada sumbu X
shearing_matrix_x = np.array([
    [1, k],
    [0, 1]
])

# Matriks transformasi untuk shearing pada sumbu Y
shearing_matrix_y = np.array([
    [1, 0],
    [k, 1]
])

# Plot hasil transformasi
fig, ax = plt.subplots()
draw_axes_and_grid(ax)
ax.scatter(points[0, :], points[1, :], color='red', label='Original')

# Shearing pada sumbu X
transformed_points_x = transform_and_plot(shearing_matrix_x, points, ax, color='blue', label=f'Shearing X (k={k})')

# Shearing pada sumbu Y
transformed_points_y = transform_and_plot(shearing_matrix_y, points, ax, color='green', label=f'Shearing Y (k={k})')

ax.legend()
plt.show()
